Quick Guide

1. Click any section in the sidebar to edit website content.
2. Go to Media Library to upload photos, then use "Choose Photo" in About or Team to assign them.
3. All changes save automatically to your browser — open index.html in the same browser to see updates live.
4. Use Enquiries to read messages submitted via the contact form.
5. Export your data anytime using the Export / Import button above.
